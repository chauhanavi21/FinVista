# FinVista

## Overview
FinVista is a personal finance management web application that helps users manage accounts,
track transactions, and monitor budgets through an interactive dashboard.

## Key Features
- Account management
- Income and expense tracking
- Transaction creation and editing
- Budget visualization
- Secure authentication
- Responsive dashboard UI

## Tech Stack
- Next.js 13, React
- Tailwind CSS, Shadcn/UI
- Clerk Authentication
- Supabase Database
- Vercel Deployment

## Setup
```bash
git clone https://github.com/chauhanavi21/FinVista.git
cd FinVista
npm install
npm run dev
```
